# [YAPEXT - yet another personal expenses tracker][repo]

Like [redline][redline] but simplified.

## Requirements

- node 20
- capacitor 7
- [vue 3.5][vue]
- [dexie 4][dexie]
- date-fns 4
- [@mdi/font 7][mdi]
- vue-data-ui 2.6

### Running this example

To run the provided example, you can use `npm start` command.

```bash
npm run dev
```
